4. Create nginx definition
	a. Get startup script and deploy in order to use service
	https://askubuntu.com/questions/257108/trying-to-start-nginx-on-vps-i-get-nginx-unrecognized-service

	https://www.digitalocean.com/community/tutorials/how-to-install-nginx-on-centos-7
	(EPEL release required for nginx)

	# Create file
	sudo nano /etc/init.d/nginx
	sudo chmod +x /etc/init.d/nginx

	# Add nginx to the system startup
	sudo chkconfig --add nginx
	sudo chkconfig --level 2345 nginx on
